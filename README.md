# Machine Learning Clustering Projects

This repository contains practical and real-world Machine Learning clustering projects using different unsupervised learning algorithms and experimentation techniques.

Each project is organized in a separate folder and includes:
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Multiple clustering models
- Hyperparameter experimentation
- Model comparison and evaluation
- Separate notebooks for each model and experiment
